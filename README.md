## Subito ##
Subito is a utility application easing tv show subtitles download.

### Usage ###
You can find help by launching 

$ <path_to_subito_directory>/bin
$ ./subito --help
or 
$ ruby subito --help


### Requirements ###
* Ruby >1.9
* Mechanize
and:
* Flexmock
* Fakeweb
 to run tests.

To install dependencies, run :
$ bundle install
